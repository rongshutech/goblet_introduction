controller设计

表单数据处理

  • url中的id数据
    • 作为形参，直接使用即可
  • query中的数据
    • 可使用cx.StrFormValue(key)来解析
    http://basePath:port/route?k=v
fmt.Println("the result is :",cx.StrFormValue("k"))

the result is :v
  • body中的数据
    • 可使用cx.Fill()来进行填充
    body:
	"{"name":"123","phone":"xxxxxxxxxxx"}"

type Body struct{
	Name string
	Phone string
}

var body Body
cx.Fill(&body)
fmt.Printf("parse body, the name is %s, the phone is %s\n",body.Name,body.Phone)

parse body, the name is 123, the phone is xxxxxxxxxxx
  • header中的数据
    • 使用cx.ReqHeader().Get(key)来解析

数据返回

  • cx.AddRespond()，进行body数据的累加，StatusCode默认为200
  • cx.Respond()
    • 默认情况，进body数据的赋值，StatusCode为200
    • error类型，进body数据的赋值，StatusCode为500
  • cx.RespondError(err error)，用err来对body数据赋值，StatusCode默认为400
  • cx.RespondWithStatus()，进行body和StatusCode两者的赋值
  • cx.RespondStatus()，进行StatusCode的赋值
