goblet安装

goblet 的安装是典型的 Go 安装包的形式：

go get github.com/extrame/goblet

常见问题：

  • git 没有安装，请自行安装不同平台的 git，如何安装请自行搜索

  • git https 无法获取，请配置本地的 git，关闭 https 验证：

    git config --global http.sslVerify false

  • 无法上网怎么安装 golbet，目前没有好的办法

goblet升级

goblet 升级分为 go 方式升级和源码下载升级：

  • Go 升级,通过该方式用户可以升级 goblet 框架，强烈推荐该方式：
    go get -u github.com/extrame/goblet

  • 源码下载升级，用户访问 https://github.com/extrame/goblet ,下载源码，然后覆盖到 $GOPATH/src/github.com/extrame/goblet 目录，然后通过本地执行安装就可以升级了：
    go install github.com/extrame/goblet

  • test.go
package main

import (
	"github.com/extrame/goblet"
)

type Test struct {
	goblet.Route  "/api/test"
	goblet.Render "json"
}

func (t *Test) ReadMany(cx *goblet.Context) {
	cx.AddRespond("hello", "world")
	cx.RespondOK()
}

func main() {
	server := goblet.Organize("test")
	server.ControlBy(&Test{})
	server.Run()

}
